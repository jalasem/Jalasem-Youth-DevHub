Materializecss Starter

Here is a starter project that brings together Materializecss (A modern responsive front-end framework based on Material Design), font-awesome, ionions and Animatecss together

Don't forget to star this project on Github if you find it useful



What do I get from this Project?

Material Design

Materializecss utilizes elements and principles of Material Design and it’s able to create a framework that incorporates components and animations that provide more feedback to users plus a single underlying responsive system across all platforms allow for a more unified user experience.
